# Tic Tac Toe (C Version)
A console-based Tic Tac Toe game written in C where the user plays against the computer. Includes two difficulty modes:
- Human (standard)
- God (hard)

## Features
- Smart AI opponent
- Score tracking
- Clean terminal UI
